# Showcase Manifest

This main message intentionally demonstrates a cohesive, product-shaped Components V2 experience.

## Included surfaces

- Rich layout composition with containers, sections, separators, thumbnails, media galleries, files, and action rows
- Message select menus for launch planning
- Modal flows using labels, text display components, text inputs, string selects, user selects, role selects, mentionable selects, channel selects, and file upload
- Stateful updates to one shared public showcase message
- Public follow-up publish actions that feel like real product outcomes

## Operating model

- The user who opens the showcase owns the live controls for that session
- Assets are attached once and only shown when a V2 component references them
- All visible message surfaces are V2 component-driven
